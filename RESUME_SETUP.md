# 履歷 PDF 設定說明

## 如何添加履歷 PDF 文件

1. 將您的履歷 PDF 文件命名為 `個人簡歷(研究所).pdf`
2. 將文件放在專案根目錄（與 `index.html` 同一層）
3. 確保文件大小適中（建議小於 2MB）

## 下載位置

網站中有兩個下載履歷的按鈕：
1. **Profile 區塊中的小按鈕**：位於個人照片下方
2. **Contact 區塊中的大按鈕**：位於聯絡資訊區頂部

兩個按鈕都會下載相同的 PDF 文件。

## GitHub Pages 路徑

如果您使用 GitHub Pages 部署，PDF 文件的路徑應該是：
- 本地開發：`個人簡歷(研究所).pdf`
- GitHub Pages：`https://yourusername.github.io/repository-name/個人簡歷(研究所).pdf`

## 測試

在瀏覽器中打開網站，點擊任一「下載完整履歷 (PDF)」或「Download Full Resume (PDF)」按鈕，應該能夠下載 PDF 文件。

如果無法下載，請檢查：
1. 文件是否正確命名為 `個人簡歷(研究所).pdf`
2. 文件是否在正確的位置（專案根目錄）
3. 文件權限是否正確
4. 瀏覽器是否允許下載文件
